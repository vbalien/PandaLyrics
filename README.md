# PandaLyrics

Spotify데스크탑앱에서 노래를 재생하면 alsong서버의 가사를 화면에 그려줍니다.



## 필요

* Spotify의 데스크탑앱
* [Spicetify](https://spicetify.app/)
* [pandaLyrics.js](https://github.com/vbalien/spicetify-extension-pandaLyrics) (프로그램 실행시 자동으로 설치)
